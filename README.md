# House Price ML Pipeline

End-to-end machine learning pipeline for house price prediction.

## Steps
- Data loading & EDA
- Missing value handling (structural vs real)
- Feature encoding (ordinal, nominal, binary)
- RandomForest model

RandomForest was chosen as a strong baseline for handling non-linear relationships without heavy tuning.

## Result
RMSE: ~28K (solid baseline for first iteration)

## Key Insight
Understanding missing data (structural vs real) is critical for building reliable ML pipelines.
